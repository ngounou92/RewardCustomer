Customer Reward Application

This web application demonstrates a customer Reward Application on blockchain using Hyperledger Fabric


A customer Reward program allows companies to reward customers who frequently make purchases. Customers are able to earn points on purchases, which can translate into some type of reward such as discount, freebie or special customer treatment. The Customers work toward a certain amount of points to redeem their reward. These programs can have multiple companies as partners on the application, to cater to a customer base. Indeed, current loyalty program systems are restraint on relations. This restraint can be removed by using blockchain technology(hyperledger faric) to join directly the customers and the companies.

This blockchain model for a customer reward program enhances the value of points to loyalty program members and brings in new value to the company by creating trusted transactions. Participants in this network have a more level relation among each other and points are in the centric position to connect all participants(customer, companies).
